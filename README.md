# 主要特性
1. 钓鱼，键盘左右移动（长按），Enter下钩；或者点屏幕左边右边移动（单点），点Jimmy下钩。
2. 飞镖，注意别打小马。
3. PvP，操作略有些繁琐：
   正常游戏分两种操作模式，wasd和上下左右。
     WASD: wasd移动，E横激光，C纵激光，Q进入防御，X开炮。
     上下左右：上下左右移动，PageUp横激光，PageDown纵激光，右Ctrl进入防御，右Alt开炮。
     单机中，Jimmy是wasd, ZDY是上下左右。
